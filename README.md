

Projeto de e-commerce front-end desenvolvido com HTML, CSS e JavaScript puro.

Funcionalidades
- Vitrine geral com todos os produtos
- Carousel de categorias
- Tela individual por categoria
- Detalhe do produto
- Carrinho com LocalStorage
- Envio de pedido via WhatsApp
- Sistema de temas



## Estrutura de telas

- Home: tela inicial de entrada

- Categorias: exibe categorias em carousel e vitrine geral de produtos

- Categoria específica: mostra apenas produtos da categoria selecionada

- Detalhe do produto: informações detalhadas e controle de quantidade

- Carrinho: lista de produtos selecionados e total.


## Decisões de UX

- Categorias ficam em carousel horizontal para facilitar navegação mobile
- Produtos aparecem juntos na vitrine geral para simular experiência de loja
- Tela de categoria é separada para foco no conteúdo
- Carrinho persiste usando LocalStorage para não perder dados.


## Tecnologias utilizadas
- HTML5
- CSS3 (Flexbox e Grid)
- JavaScript Vanilla
- LocalStorage
- GitHub Pages.
